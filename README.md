## 安装步骤

1. 克隆仓库：
    ```sh
    git clone https://github.com/0xbaiwan/arichain_bot
    cd arichain
    ```

2. 安装依赖：
    ```sh
    npm install
    ```

3. 可选使用代理：
- 将代理粘贴到 `proxy.txt` 中，格式为 `http://用户名:密码@ip:端口`
    ```sh
    nano proxy.txt
    ```

4. 运行脚本：
    ```sh
    npm run start
    ```

 所有账户信息保存在 `accounts.txt` 中
